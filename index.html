对象的方法

对象的方法

数组对象的方法

  • .push(): 向数组末尾添加一个或多个元素，并返回新的长度。
  • .pop(): 移除数组最后一个元素并返回该元素的值。
  • .shift(): 移除数组的第一个元素并返回该元素的值，同时将数组长度减一。
  • .unshift(): 在数组的开头添加一个或多个元素，并返回新的长度。
  • .splice(): 从数组中添加或移除元素。
  • .slice(): 返回一个新的数组，包括从开始到结束（不含结束）部分的元素。
  • .concat(): 用于连接两个或多个数组。
  • .sort(): 用于对数组的元素进行排序。
  • .reverse(): 用于颠倒数组中元素的顺序。
  • .indexOf(): 返回数组中某个元素第一次出现的索引。
  • .forEach(): 对数组的每个元素执行一次提供的函数。

字符串对象方法

  • .length: 返回字符串的长度。
  • .charAt(): 返回指定位置的字符。
  • .charCodeAt(): 返回给定索引处字符的 Unicode 值。
  • .concat(): 用于连接两个或多个字符串。
  • .indexOf(): 返回指定值在字符串中首次出现的位置。
  • .lastIndexOf(): 返回指定值在字符串中最后一次出现的位置。
  • .slice(): 提取字符串的一部分并返回新字符串。
  • .substring(): 返回位于两个指定下标之间的字符。
  • .toUpperCase(): 将字符串转换为大写字母。
  • .toLowerCase(): 将字符串转换为小写字母。
  • .replace(): 用一个字符串替换另一个字符串。
  • .split(): 将字符串分割成字符串数组。
  • .trim(): 去除字符串两端的空白字符。

数字对象的方法

  • Number.parseFloat(): 将字符串解析为浮点数并返回。
  • Number.parseInt(): 将字符串解析为整数并返回。
  • Number.isNaN(): 判断一个值是否为 NaN。
  • Number.isFinite(): 判断一个值是否为有限数。
  • Number.isInteger(): 判断一个值是否为整数。

Date对象的方法

  • getDate(): 获取一个月中的某一天（1 到 31）。
  • getDay(): 获取星期几（0 表示星期日，1 表示星期一，以此类推）。
  • getMonth(): 获取月份（0 表示一月，11 表示十二月）。
  • getFullYear(): 获取年份（四位数）。
  • getHours(): 获取小时（0 到 23）。
  • getMinutes(): 获取分钟（0 到 59）。
  • getSeconds(): 获取秒数（0 到 59）。
  • getMilliseconds(): 获取毫秒（0 到 999）。
  • getTime(): 获取从 1970 年 1 月 1 日至今的毫秒数。
  • toString(): 返回日期的字符串表示。

RegExp 对象方法

  • test(): 测试字符串是否匹配正则表达式，并返回 true 或 false。
  • exec(): 在字符串中执行正则表达式搜索，返回包含匹配内容的数组或 null。
  • compile(): 编译正则表达式，用于在多次执行中重复使用该正则表达式。

Function 对象方法

  • apply(): 调用函数并指定 this 值和参数，参数以数组形式传入。
  • call(): 调用函数并指定 this 值和参数，参数以参数列表形式传入。
  • bind(): 创建一个新函数，其中 this 值由指定的对象提供。

Math 对象方法

  • Math.abs(): 返回一个数的绝对值。
  • Math.round(): 把一个数四舍五入为最接近的整数。
  • Math.floor(): 对一个数进行向下取整。
  • Math.ceil(): 对一个数进行向上取整。
  • Math.max(): 返回一组数中的最大值。
  • Math.min(): 返回一组数中的最小值。
  • Math.pow(): 返回一个数的指定次幂。
  • Math.sqrt(): 返回一个数的平方根。
  • Math.random(): 返回一个大于等于 0 小于 1 的随机数。

对象——原始值转换

hint

对象到原始值的转换，是由许多期望以原始值作为值的内建函数和运算符自动调用的。

这里有三种类型（hint）：

  • "string"（对于 alert 和其他需要字符串的操作）

  • "number"（对于数学运算）

  • "default"（少数运算符，通常对象以和 "number" 相同的方式实现 "default" 转换）

    转换规则如下：

  • 首先检查对象是否具有 Symbol.toPrimitive 方法，如果有，则调用该方法并传入运算符或打印的 hint。该方法必须返回一个原始值。

  • 如果对象没有 Symbol.toPrimitive 方法，则根据 hint 类型进行转换：

    • 对于 hint 为 “string”，JavaScript 尝试调用对象的 toString 方法，如果没有则调用 valueOf 方法。它们必须返回一个原始值。
    • 对于 hint 为 “number” 或 “default”，JavaScript 尝试调用对象的 valueOf 方法，如果没有则调用 toString 方法。它们必须返回一个原始值。

    • symbol 用于唯一的标识符。
  • 以及一种非原始数据类型：
    • object 用于更复杂的数据结构。

undefined 不应该被与其他值进行比较：

总结

  • 比较运算符始终返回布尔值。
  • 字符串的比较，会按照“词典”顺序逐字符地比较大小。
  • 当对不同类型的值进行比较时，它们会先被转化为数字（不包括严格相等检查）再进行比较。
  • 在非严格相等 == 下，null 和 undefined 相等且各自不等于任何其他的值。
  • 在使用 > 或 < 进行比较时，需要注意变量可能为 null/undefined 的情况。比较好的方法是单独检查变量是否等于 null/undefined。

运算符

JavaScript 支持以下运算符：

  • 算数运算符

    常规的：+ - * /（加减乘除），取余运算符 % 和幂运算符 **。二进制加号 + 可以连接字符串。如果任何一个操作数是一个字符串，那么另一个操作数也将被转换为字符串：alert( '1' + 2 ); // '12'，字符串 alert( 1 + '2' ); // '12'，字符串

  • 赋值

    简单的赋值：a = b 和合并了其他操作的赋值：a *= 2。

  • 按位运算符

    按位运算符在最低位级上操作 32 位的整数：详见 文档。

  • 三元运算符

    唯一具有三个参数的操作：cond ? resultA : resultB。如果 cond 为真，则返回 resultA，否则返回 resultB。

  • 逻辑运算符

    逻辑与 && 和或 || 执行短路运算，然后返回运算停止处的值（true/false 不是必须的）。逻辑非 ! 将操作数转换为布尔值并返回其相反的值。

  • 空值合并运算符

    ?? 运算符从一列变量中，选取值为已定义的值（defined value）的变量。a ?? b 的结果是 a，除非 a 为 null/undefined，这时结果是 b。

  • 比较运算符

    对不同类型的值进行相等检查时，运算符 == 会将不同类型的值转换为数字（除了 null 和 undefined，它们彼此相等而没有其他情况），所以下面的例子是相等的：alert( 0 == false ); // true alert( 0 == '' ); // true其他比较也将转换为数字。严格相等运算符 === 不会进行转换：不同的类型总是指不同的值。值 null 和 undefined 是特殊的：它们只在 == 下相等，且不相等于其他任何值。大于/小于比较，在比较字符串时，会按照字符顺序逐个字符地进行比较。其他类型则被转换为数字。

  • 其他运算符

    还有很少一部分其他运算符，如逗号运算符。
